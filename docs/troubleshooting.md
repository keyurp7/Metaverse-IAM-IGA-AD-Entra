# Troubleshooting

# Scope

The lab includes troubleshooting across identity integration, runtime support, and platform behavior.

# Areas Covered
- LDAP and LDAPS validation
- DNS and directory connectivity checks
- certificate validation and Java truststore troubleshooting
- SailPoint connector validation
- IQService and Connector Gateway runtime checks
- Tomcat runtime issues
- HTTP 405 platform behavior investigation
- Entra Connect sync health visibility

# Why It Matters

A useful IAM lab should show not only successful configuration, but also the ability to isolate, validate, and resolve issues across multiple layers of the stack.
