## Before Opening an Issue

1. **Search existing issues** to see if it's already been reported
2. **Check the documentation** — your answer might already be in the [docs](docs/) folder
3. **Be specific** — include screenshots, links, or steps to reproduce where possible

## What This Project Covers

This is a lab and portfolio project. It demonstrates IAM concepts, architecture, and governance workflows — it is not a production application or a supported product. Questions about general IAM concepts, SailPoint, Active Directory, Entra ID, or the lab setup are all welcome.
